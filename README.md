# 🚀 Vendas com IA & Tech Sales - Bootcamp Core

Repositório dedicado ao armazenamento de projetos, desafios práticos e mentorias desenvolvidos durante o Bootcamp focado na intersecção entre **Inteligência Artificial, Automação Comercial e Estratégias de Negócio**.

O objetivo deste espaço é consolidar o conhecimento em engenharia de prompts, fluxos de automação de processos comerciais (H2H - Human to Human) e aplicação prática de IA generativa para otimização de funis de vendas e gestão de produtos digitais.

---

## 📌 Visão Geral do Projeto

Este repositório serve como portfólio técnico, demonstrando a evolução na aplicação de tecnologias emergentes para resolver dores reais de negócios (ganho de eficiência, produtividade e eliminação de tarefas manuais repetitivas).

### 🛠️ Tecnologias & Ferramentas Focadas
* **IA Generativa:** Engenharia de Prompts (ChatGPT, Microsoft Copilot)
* **Automação de Processos:** Estruturação de fluxos lógicos e integração
* **Banco de Dados & Dados:** Consultas estruturadas (SQL) auxiliadas por IA
* **Desenvolvimento Front-end:** Fundamentos de React, Arquitetura de Componentes e Frameworks Responsivos

## 🚀 Desafios Concluídos

### 🧩 1. Desafio Criativo: Simplificando Tarefas com IA
* **Contexto:** Automação do primeiro contato e qualificação de leads frios.
* **Solução:** Criação de um prompt estruturado (Intenção + Contexto + Regras) focado na abordagem **H2H (Human to Human)**, conectando ferramentas de automação à API de IA generativa para gerar copys personalizadas e dinâmicas, reduzindo o tempo operacional do time comercial.
* **Pasta do Projeto:** `/desafios-criativos/produtividade-ia/`

### 📊 2. Escrevendo Consultas SQL com Microsoft Copilot
* **Contexto:** Uso de copilotos de inteligência artificial para ganho de velocidade em análise de dados.
* **Solução:** Utilização do Microsoft Copilot para estruturar e acelerar a escrita de consultas SQL de bancos de dados comerciais, unindo tecnologia à tomada de decisão baseada em dados.
